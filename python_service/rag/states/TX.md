# TEXAS TAX RULES 2025
# No State Income Tax

## OVERVIEW
**TEXAS HAS NO STATE INCOME TAX**

This is protected by the Texas Constitution.

## WHAT'S NOT TAXED
- Wages - NOT TAXED
- Self-Employment - NOT TAXED
- Interest/Dividends - NOT TAXED
- Capital Gains - NOT TAXED
- Social Security - NOT TAXED
- Retirement Income - NOT TAXED

## NO STATE RETURN NEEDED
You only file federal taxes. No Texas state return.

## OTHER TAXES
- Sales Tax: 6.25% state + up to 2% local = 8.25% max
- Property Tax: ~1.8% (varies by county, among highest in US)

## COMMON QUESTIONS

Q: Does Texas have income tax?
A: No, Texas has NO state income tax.

Q: Do I file a Texas return?
A: No, there is no state income tax return needed.

Q: Is Social Security taxed in Texas?
A: No, Texas has no income tax so nothing is taxed at state level.

Q: Is retirement taxed in Texas?
A: No, all retirement income is tax-free at the state level.
