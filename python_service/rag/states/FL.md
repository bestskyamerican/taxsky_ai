# FLORIDA TAX RULES 2025
# No State Income Tax

## OVERVIEW
**FLORIDA HAS NO STATE INCOME TAX**

Protected by the Florida Constitution.

## WHAT'S NOT TAXED
- All income - NOT TAXED
- Social Security - NOT TAXED
- Retirement - NOT TAXED
- Capital Gains - NOT TAXED

## NO STATE RETURN NEEDED
You only file federal taxes.

## WHY RETIREES LOVE FLORIDA
- No tax on Social Security
- No tax on pensions
- No tax on 401k/IRA distributions
- No estate/inheritance tax

## COMMON QUESTIONS

Q: Does Florida have income tax?
A: No, Florida has NO state income tax.

Q: Is retirement taxed in Florida?
A: No, all retirement income is tax-free at state level.
